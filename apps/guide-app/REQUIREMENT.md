# Guide App — 서비스 요구사항 (REQUIREMENT)

> 풋살 팀 매칭 앱. 팀을 만들거나 가입하고, 다른 팀과 경기를 잡을 수 있다.

---

## 1. 서비스 개요

| 항목 | 내용 |
|------|------|
| 서비스명 | Guide App (풋살 팀 매칭) |
| 대상 사용자 | 풋살을 즐기는 일반인 |
| 핵심 가치 | 팀 결성 → 경기 매칭까지 한 앱에서 |
| 플랫폼 | 모바일 웹 (max-width 480px) |

---

## 2. 사용자 역할

| 역할 | 조건 | 가능한 행동 |
|------|------|-------------|
| **비회원** | 미로그인 | 회원가입, 로그인만 가능 |
| **일반 유저** | 로그인 + 팀 미소속 | 팀 검색, 가입 신청, 프로필 편집 |
| **멤버** | 로그인 + 팀 소속 | 내 팀 조회, 경기 신청, 팀 탈퇴 |
| **캡틴** | 팀 생성자 or 권한 이양 받은 자 | 팀 관리, 가입 신청 수락/거절, 경기 등록/취소 |

> 한 유저는 동시에 하나의 팀에만 소속될 수 있다.

---

## 3. 인증 / 계정

### 3-1. 회원가입
- 이메일, 비밀번호(8자 이상), 닉네임(2~20자) 입력
- 이메일 중복 시 에러 표시

### 3-2. 로그인
- 이메일 + 비밀번호 인증
- 성공 시 Access Token(메모리) + Refresh Token(httpOnly 쿠키) 발급
- AT 만료 시 RT 쿠키로 자동 갱신 (Hybrid Refresh Token Rotation)
- 비로그인 상태에서 보호 경로 접근 → `/login` 리다이렉트

### 3-3. 로그아웃
- 세션 삭제, AT 초기화, `/login` 이동

### 3-4. 프로필 편집 (`/profile`)
- 닉네임 수정
- 포지션 선택: FW / MF / DF / GK (선택 사항)
- 실력 레벨 선택: 1~5 (선택 사항, 기본값 3)
- 이메일은 읽기 전용 (변경 불가)

## 4. 팀 (`/team`)

화면은 "내 팀" / "팀 찾기" 두 탭으로 구성된다.

### 4-1. 팀 생성
- 팀 이름(필수, 2~30자), 소개(선택, 200자 이하), 로고 URL(선택) 입력
- 생성자는 자동으로 해당 팀의 **캡틴**이 된다
- 팀 생성과 동시에 `team_members`에 CAPTAIN 역할로 등록된다 (트랜잭션)

### 4-2. 팀 찾기 / 검색
- 팀 이름으로 검색 (대소문자 무시, 부분 일치)
- 검색어 없으면 전체 팀 목록 반환
- 각 팀 카드에 팀명, 소개, 현재 멤버 수 표시

### 4-3. 가입 신청
- 팀 카드 클릭 → 가입 신청 화면 (가입 메시지 입력 가능, 선택)
- **이미 다른 팀 소속**이면 신청 불가 (에러 토스트)
- 동일 팀에 중복 신청 불가 (PENDING 상태 존재 시 에러)
- 거절(REJECTED)된 이력이 있으면 재신청 가능

### 4-4. 가입 신청 처리 (캡틴 전용)
- 내 팀 탭에서 PENDING 신청 목록 확인
- 신청자 닉네임, 포지션, 실력 레벨 확인 가능
- **수락**: `team_members`에 MEMBER로 추가 (트랜잭션)
- **거절**: 신청 상태만 REJECTED로 변경, 팀원 미추가

### 4-5. 내 팀 조회
- 팀 이름, 소개, 나의 역할(캡틴/멤버) 표시
- 전체 팀원 목록 (닉네임, 포지션, 실력, 역할)

### 4-6. 팀 탈퇴
- 멤버만 가능, **캡틴은 탈퇴 불가**
- 탈퇴 전 확인 다이얼로그 표시

## 5. 경기 (`/matches`)

화면은 "경기 찾기" / "내 팀 경기" 두 탭으로 구성된다.

### 5-1. 경기 등록 (캡틴 전용)
- 경기 날짜/시간(필수), 장소(필수, 2~100자), 메모(선택, 200자 이하) 입력
- 등록 팀 = 홈팀, 초기 상태 = **OPEN**
- 캡틴이 아닌 유저가 시도하면 에러

### 5-2. 경기 목록 (경기 찾기 탭)
- **OPEN** 상태 경기만 표시
- 경기 날짜 오름차순 정렬
- 각 카드에 홈팀명, 날짜/시간, 장소, 메모, 상태 배지 표시
- 카드 클릭 → 경기 신청 화면

### 5-3. 경기 신청 (캡틴 전용)
- 신청한 팀 = 어웨이팀
- **선착순**: 신청 즉시 상태 OPEN → **MATCHED**, 별도 수락 없음
- 자기 팀이 등록한 경기에는 신청 불가 (SAME_TEAM 에러)
- OPEN이 아닌 경기 신청 시 에러

### 5-4. 내 팀 경기 (내 팀 경기 탭)
- 내가 속한 팀의 홈/어웨이 경기 모두 표시
- 상태별 표시:
  - **OPEN**: 홈팀 캡틴에게 "취소" 버튼 표시
  - **MATCHED**: 홈팀 vs 어웨이팀명 표시
  - **COMPLETED**: 완료 배지
  - **CANCELLED**: 취소 배지

### 5-5. 경기 취소 (홈팀 캡틴 전용)
- OPEN 또는 MATCHED 상태 경기만 취소 가능
- 취소 전 확인 다이얼로그 표시
- 취소 시 상태 → **CANCELLED**
- COMPLETED/CANCELLED 경기는 취소 불가

---

## 6. 경기 상태 흐름

```
OPEN ──→ MATCHED (경기 신청 성공)
     └──→ CANCELLED (홈팀 캡틴이 취소)

MATCHED ──→ COMPLETED (경기 완료, 현재 미구현)
        └──→ CANCELLED (홈팀 캡틴이 취소)
```

---

## 7. 팀 가입 상태 흐름

```
없음 ──→ PENDING (가입 신청)
              ↓
       ACCEPTED ──→ TeamMember 등록
       REJECTED ──→ 재신청 가능
```

---

## 8. 네비게이션 구조

```
/ (비로그인) → /login 리다이렉트

/login
/signup

(로그인 후)
/dashboard   홈 — 로그아웃 버튼, 팀/경기 바로가기
/team        팀 — 탭: 내 팀 | 팀 찾기
/matches     경기 — 탭: 경기 찾기 | 내 팀 경기
/profile     프로필 편집
```

하단 탭바로 4개 페이지 이동.

## 9. 제약 조건 및 비즈니스 규칙

| 규칙 | 내용 |
|------|------|
| 다중 팀 소속 | 불가 — 유저는 최대 1개 팀 |
| 캡틴 탈퇴 | 불가 (에러 반환) |
| 경기 등록 | 캡틴만 가능 |
| 경기 신청 | 캡틴만 가능, 자기 팀 홈 경기 신청 불가 |
| 경기 취소 | 홈팀 캡틴만 가능 |
| 가입 신청 처리 | 팀 캡틴만 가능 |
| 가입 신청 중복 | PENDING 상태 중 재신청 불가 |
| 팀 삭제 | 현재 미지원 |
| 캡틴 이양 | 현재 미지원 |
| 경기 완료 처리 | 현재 미지원 (상태만 존재) |

## 10. 현재 미구현 (향후 과제)

| 기능 | 설명 |
|------|------|
| 캡틴 이양 | 팀 내 다른 멤버에게 캡틴 역할 이전 |
| 멤버 추방 | 캡틴이 특정 멤버를 팀에서 제거 |
| 팀 삭제 | 캡틴이 팀을 해산 |
| 경기 완료 처리 | 경기 결과 입력 및 COMPLETED 상태 전환 |
| 경기 정보 수정 | OPEN 상태 경기의 날짜/장소 수정 |
| 경기 필터 | 지역/날짜 기준 경기 목록 필터링 |
| 푸시 알림 | 가입 신청/수락/거절 시 알림 |
| 팀 로고 업로드 | URL 입력 대신 이미지 직접 업로드 |
